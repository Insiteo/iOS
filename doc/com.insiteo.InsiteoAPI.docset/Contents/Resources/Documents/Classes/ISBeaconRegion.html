Parameters

proximityUUID

The unique ID of the beacon being targeted. This value must not be nil.

identifier

A unique identifier to associate with the returned region object. You use this identifier to differentiate regions within your application. This value must not be nil.

label

Custom Beacon Region label.

message

Custom Beacon Region message.

Timer used to limit on beacon region entry notification if user is in the same beacon region.

The site ID associated with the beacon region.
